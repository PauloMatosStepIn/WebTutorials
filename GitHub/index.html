GitHub Tutorials : Projecto Local

Para a criação dum projecto GIT no computador usando VsCode:
- Abrir o vscode
- Abrir o Terminal (no Teclado Português: Ctrl+Shift+Ç ou Ctrl+Shif+P para lista de comandos, e escrver "terminal" para obter lista de comandos relacionados, e escolher: "Open New External Terminal" )
- No Terminal do VsCode ir para a directoria do projecto
- Executar: code . e passar a usar a nova janela.
- Executar: git --version (para verificar que o GIT está instalado)
- Executar: git init (transforma a directoria num Repositório GIT Vazio)
- Executar: git add . (adiciona ficheiros ao Repositório GIT)
- Executar: git commit -a -m "first commit" (Passa ficheiros da "Área de trabalho p/Repositório GIT Local")
- Executar: git config user.name "<Your Name>"
- Executar: git config user.email "<Your Email>" (de forma a identificar autor)
