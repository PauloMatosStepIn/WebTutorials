Generating a new SSH key

No Terminal VsCode abrir uma sessão "Git Bash" (com os icons do canto superior direito do terminal)

Executar: ls -al ~/.ssh (verificar se existem os ficheiros de certificados id_ed25519 e id_ed25519.pub)

Executar: ssh-keygen -t ed25519 -C "<Your email>" (Generating a new SSH key)

Executar: eval "$(ssh-agent -s)" (Ensure the ssh-agent is running)

Executar: ssh-add ~/.ssh/id_ed25519 (Add your SSH private key to the ssh-agent)

Executar: clip < ~/.ssh/id_ed25519.pub (Copy the SSH public key to your clipboard.)


Adding a new SSH key to your GitHub account

Aceder à sua conta no GitHub, Clicar na foto no canto superior direito p/aceder ao menu do utilizador.

Escolher Settings

No Menu do lado esquerdo, escolher SSH & GPG keys

Em SSH keys, clicar em New SSH key

Fazer "paste" da chave pública para o campo "key", e confirmar clicando em: "Add SSH key"
