GitHub Tutorials : Repositório GitHub

Para a criação dum Repositório GitHub:

- Abrir o gitHub no browser
- Fazer "Sign In" na sua conta, caso não tenha, efectuar primeiro "Sign Up"
- Ir para "Repositories", clicar em "NEW"
- Preencher "Repository name"
- Preencher "Description"
- Escolher "Public" (caso pretenda que fique visível publicamente) ou "Private" (caso pretenda manter o conteúdo apenas para si)
- Clicar em: "Create repository"

- Na página de "Quick Setup" escolher a opção SSh, e copiar a linha que começa com: "git remote add origin git@github.com:"
- Ir para o terminal do VsCode do Projecto criado localmente e executar a linha copiada.

- Opcionalmente, na página do repositório Clicar na opção Code, escolher SSH,
copiar o endereço que começa em: git@github.com e excutar no Terminal do VsCode o comando:
git remote add origin <string copiada>

- Executar: "git push origin master" (p/actualizar GitHub Repo com a versão local)
(Verificar conteúdo no GitHub)

De forma a evitar ter de dar o comando:
"git push origin master"
Faça o Set up branch 'master' to track 'origin/master' executando o comando:
- "git push --set-upstream origin master"

Assim, pode passar a executar apenas o comando:
"git push"




Dica: para abrir o Terminal no VsCode pode usar a combinação de teclas: Ctrl-Shift-Ç (Num teclado Português)
