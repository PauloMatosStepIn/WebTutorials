GitHub Tutorials : GitHub Secrets for Actions

Na sua conta de GitHub aceder ao repositório, no menu do cabeçalho escolher a opção Settings

No menu do lado esquerdo escolher Secrets > Actions

Clicar no botão do lado direito New repository secret

Preencher Name e Value e confirmar em "Add secret"

Estes secrets podem ser utilizados em ficheiros yaml da seguinte forma: ${{ secrets.FTP_PASSWORD }} onde FTP_PASSWORD é neste caso o nome do "secret".
