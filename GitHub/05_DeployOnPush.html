GitHub Tutorials : CI/CD Deploy On Push

Na sua conta de GitHub aceder ao repositório, no menu do cabeçalho escolher a opção Actions

Clicar em set up a workflow yourself

Copiar o conteúdo do main.yml para o seu ficheiro yaml.

Escolher a opção de Start Commit, preencher descrição e confirmar com Commit Changes

(Ir a Actions e ver os resultados da execução)

(após a execução da actualização o seu site deve estar actualizado com o novo conteúdo)

Como requisito é necessário configurar um utilizador de ftp com password e o path para o directório adequado no WebServer. É necessário também criar os seguintes "Action Secrets" no repositório: FTP_SERVER, FTP_USER e FTP_PASSWORD (ver opção GitHub Secrets for Actions para informação de como criar).
