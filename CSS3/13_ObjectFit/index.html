Web CSS3 Tutorials : Object Fit

A propriedade CSS object-fit é usada para especificar como uma tag <img> ou <video> se deve redimensionar de forma a preencher a área onde está definida(container). Esta propriedade especifica como o conteúdo deve preencher o container, fazendo-o de diversas formas: preservando a proporção da imagem, esticando ou encolhendo a imagem de forma a preencher a área do container.

object-fit: none;
HTML
        <div class="janela">
          <img src="imagens/GirlOnPc600x314.jpg" />
        </div>
CSS
      <style>
      div.janela {
        width: 80%;
        height: 400px;
        margin: 10px auto;
        border: 1px solid #000;
      }

      div.janela img {
        width: 100%;
        height: 100%;
        object-fit: none;
      }
      </style>
