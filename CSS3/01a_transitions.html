Web CSS3 Tutorials : Transitions rotateY

Passe o Mouse por cima da área branca...

HTML
        <div class="box"></div>
CSS
      <style>
      .box {
        background: white;
        width: 300px;
        height: 300px;
        position: relative;
        margin: auto;
        top: 5px;
        /* transition-property: background, border-radius, transform; */
        transition-property: all;
        transition-duration: 1s;
        /* transition-delay: 2s, 4s; */
        transition-timing-function: ease-in;
      }
      
      .box:hover {
        background: red;
        border-radius: 50%;
        transform: rotateY(180deg);
      }
      </style>
