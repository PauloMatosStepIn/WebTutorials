Web CSS3 Tutorials : Media Types (screen/print)

PRINT this page to see the diference between media="screen" and media="print"

No media="print" é suprimida a secção lateral, Os títulos passam a preto, o texto dos artigos passa a "justificado" ocupando toda a largura, e as imagens não são apresentadas. A Imagem do Topo é substituida por texto.

No media indicado para "mobile"(min-width:320px/max-width:430px) a apresentação passa a uma coluna, imagens centradas, texto justificado e ampliado. Adaptada a apresentação do cabeçalho.
