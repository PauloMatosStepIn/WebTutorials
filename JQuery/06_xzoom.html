JQuery Tutorials : Zoom fora do Objecto

Nesta página é usado o plugin jQuery "xZoom"

Necessário fazer download do "xZoom". Para o código seguinte funcionar foi necessário: criar as sub-directorias xzoom\js (para onde foram movidos os ficheiros js da dir "dist" do download) e xzoom\css (para onde foram movidos os ficheiros css da dir "dist" do download)
