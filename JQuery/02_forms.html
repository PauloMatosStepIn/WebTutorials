fadeIn / fadeOut

Quando um campo recebe o FOCUS é mostrada a mensagem de apoio, quando perde o FOCUS a mensagem é escondida. AutoComplete colocado em Off.

Digite o seu nome completo Digite um seu email Digite apenas números
CSS
          <style>
            input[type="text"] {
              border: 1px solid #999;
              padding: 10px;
              margin-bottom: 20px;
              border-radius: 3px;
              width: 200px;
            }
      
            label,
            input[type="submit"] {
              display: block;
              margin-bottom: 3px;
            }
      
            input + span {
              margin-left: 15px;
              border: 1px solid #ccc;
              background-color: #f0f0f0;
              padding: 5px 10px;
              color: #999;
              border-radius: 3px;
              font-size: 14px;
            }
          </style>
JAVASCRIPT
          <script>
            $(function () {
              //para não mostrar conteúdos anteriores
              $("input").attr("autocomplete", "off");
    
              //optional sintax:
              //$("input[type='text'] + span").css("display","none");
              $("input[type='text'] + span").css({ display: "none" });
    
              $("input[type='text']").focus(function () {
                $(this).next().fadeIn(1000);
              });
              $("input[type='text']").blur(function () {
                $(this).next().fadeOut(1000);
              });
            });
          </script>

disabled / checked

Quando se clica na secção, habilita só os checkboxes da secção seleccionada.

Quando se clica, coloca em false todas as checkboxes.

CSS
          <style>

              height: 120px;
              box-shadow: 0 0 4px #ccc;
              background-color: #fff;
            }

            #form_sports label {
              float: left;
              width: 180px;
              display: block;
              margin-bottom: 5px;
            }

            #form_sports label.sport {
              border-bottom: 1px dotted #ccc;
              padding-bottom: 10px;
              margin-bottom: 10px;
            }
          </style>

JAVASCRIPT
            <script>
            $(function () {
              $("section.sports input:checkbox").attr("disabled", "disabled");
  
              $("section.sports input:radio").click(function () {
                //remove opção disabled
                $("section.sports input:checkbox").removeAttr("disabled");
  
                //desabilitar os outros
                //siblings -> irmaos
                $(this).parents(".sports").siblings(".sports").find(":checkbox").attr("disabled", "disabled");
  
                $("section.sports input:checkbox").prop("checked", false);
              });
            });
            </script>

        <section>

          <label><input type="checkbox" class="opcoes" />Cristiano</label>
          <label><input type="checkbox" class="opcoes" />João </label>
          <label><input type="checkbox" class="opcoes" />Joca</label>

        </section>
      </form>
JAVASCRIPT
    <script>
      $(function () {
        $(".selectall").click(function () {
          if ($(this).is(":checked")) {
            $(".opcoes").prop("checked", true);
          } else {
            $(".opcoes").prop("checked", false);
          }
        });
        $(".opcoes").click(function () {
          $(".selectall").prop("checked", false);
        });
      });
    </script>
