JQuery Tutorials : Select Box User Friendly

Nesta página é usado o plugin jQuery "Chosen (v1.8.7)"

Chosen is a jQuery plugin that makes long, unwieldy select boxes much more user-friendly.

Necessário fazer download do "chosen". Para o código seguinte funcionar foi necessário copiar o download para uma sub-directoria chosen, e foram criadas as sub-directorias chosen\js (para onde foram movidos os ficheiros js da root do download) e chosen\css (para onde foram movidos os ficheiros css e png da root do download)

Select simples c/pesquisa

JAVASCRIPT
          <script>
            $("#selectsimples").chosen();
          </script>
HTML
          <link href="chosen/css/chosen.css" rel="stylesheet" />

          <script src="external/jquery/jquery.js"></script>
          <script src="jquery-ui.js"></script>
      
          <script src="chosen/js/chosen.jquery.js"></script>
          
          <select id="selectedsimples" data-placeholder="Choose a Country..." >
            <option value=""></option>
            <option value="United States">United States</option>
            <option value="United Kingdom">United Kingdom</option>
            <option value="Afghanistan">Afghanistan</option>
            ...
            <option value="Zambia">Zambia</option>
            <option value="Zimbabwe">Zimbabwe</option>
          </select>

Select agrupado c/pesquisa por grupo ou item

JAVASCRIPT
    <script>
      $("#selectagrupado").chosen();
    </script>
HTML
          <link href="chosen/css/chosen.css" rel="stylesheet" />

          <select id="selectagrupado" data-placeholder="Escolha Equipa de Futebol Americano Favorita..." style="width: 400px;">
            <option value=""></option>
            <optgroup label="NFC EAST">
              <option>Dallas Cowboys</option>
              <option>New York Giants</option>
              <option>Philadelphia Eagles</option>
              <option>Washington Redskins</option>
            </optgroup>
            ...
          </select>

          <script src="external/jquery/jquery.js"></script>
          <script src="jquery-ui.js"></script>
      
          <script src="chosen/docsupport/jquery-3.2.1.min.js" type="text/javascript"></script>
          <script src="chosen/js/chosen.jquery.js"></script>
          <script src="chosen/docsupport/prism.js" type="text/javascript" charset="utf-8"></script>
          <script src="chosen/docsupport/init.js" type="text/javascript" charset="utf-8"></script>

Select múltiplo

Parametrizado para até 3 selecções, com aviso

JAVASCRIPT
    <script>
      $("#select3").chosen({ max_selected_options: 3 });
      $("#select3").bind("chosen:maxselected", function(){
        alert("Já seleccionou 3 Países, Obrigado.")
      });
    </script>
HTML
          <link href="chosen/css/chosen.css" rel="stylesheet" />
          
          <select id="select3" data-placeholder="Escolher até 3 Paises..." multiple style="width: 500px">
            <option value=""></option>
            <option value="United States">United States</option>
            <option value="United Kingdom">United Kingdom</option>
            <option value="Afghanistan">Afghanistan</option>
            <option value="Aland Islands">Aland Islands</option>
            ...
          </select>

          <script src="chosen/js/chosen.jquery.js">
